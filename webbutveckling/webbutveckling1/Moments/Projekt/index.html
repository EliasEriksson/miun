Minesweeper AI

Minesweeper AI is an AI built to solve googles minesweeper game. When first started the AI looks for a minesweeper board on screenshots of the screen using a simple image processing algorithm. The AI then takes control over the mouse to start sovereign the board. Between each state of the board a new screenshot is taken and to update the internal representation of the board. Each of these screenshots is scanned square by square using googles tesseract OCR engine to read the numbers from the board. The showcase here is sped up footage the average clear time on a large game board is around 110 seconds. Video of real time execution can be found in the repository .

Codescord

Codescord is a discord bot that compiles/runs code within a code block in a discord server. The codescord client is built upon another client that is starting up docker containers and sends text data to the server that lives inside of the container. The server inside of the container then compiles / runs the source code of any supported language. the stdout/stderr will be sent back to the client and discord client to be printed in the chat channel where the code block initially was posted. If the container takes too long to respond (could be that it ran malicious code, infinite loop etc) the container is killed off. The client and server internally used by codescord could be reused for other front ends where remote code execution is desired.

PCT

PCT (Python Compile Tool) is something i developed to compile cPython from source and is today my go-to method of doing so. Its a CLI program that takes version of python to compile and a python executable. The given version number of python is downloaded from python.org. The python executable is optional but will default to python3. The program will open a python shell of the given executable and get all the compile flags the given executable was compiled with and will use those flags + --enable-optimizations --with-lto - --without-ensurepip to compile the downloaded version. This is optional and will not be done if None is given instead of an executable.

IpBlacklister

IpBlacklister is s small background process that scans through the latest apache.log file for connections made from ip addresses thought to be malicious. It uses the AbuseIPDB api and if AbuseIPDBs confidence is high enough to be considered abusive the ip is blocked access through the firewall (UFW). In the image there are a few entries in the apache.log and five out of the seven non local addresses got caught by IpBlacklister.
